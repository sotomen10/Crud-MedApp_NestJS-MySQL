# Crud MedApp (Clinic Appointment Management)

## Company
**MedApp Clinic**

## Context
MedApp Clinic is a private clinic seeking to improve its appointment management through an API that allows for efficient handling of appointments, patients, doctors, and availability. The solution aims to address issues such as appointment duplication, lack of visibility into doctor availability, and difficulty tracking patient appointment history.

## Project Setup

To install the necessary dependencies, run:

```bash
$ npm install
```

## Compile and Run the Project

You can start the project in different modes:

```bash
# Development mode
$ npm run start

# Watch mode (auto-reload)
$ npm run start:dev

# Production mode
$ npm run start:prod
```

Once the project is running, you can access the following:

- The app will be running on `http://localhost:${PORT}`
- Swagger API documentation is available at `http://localhost:${PORT}/api`
- To authenticate with Google, visit `http://localhost:${PORT}/auth/google`
